The Obvious NDA

The Obvious NDA is a boring mutual nondisclosure agreement for busy companies.

The most recent version is 1.0.0. You can download it for free in Microsoft Word, Adobe Acrobat, and Real Text formats.

If your company is tired of reviewing NDAs, or signing NDAs without reading, consider using The Obvious NDA instead.

Questions and Answers

What's so special about The Obvious NDA?

Nothing. It's bone-stock. Orthodox. Unsurprising. Routine. Banal. Boring.

The Obvious NDA differs from other NDAs only in how it protects you from having to reread it every time. Every copy of The Obvious NDA comes with a certificate from the side proposing it, guaranteeing that the terms are exactly the same as those published on this site. The license for the terms requires those who make changes to drop the name “The Obvious NDA” from their changed versions.

Will The Obvious NDA improve over time?

Yes. New versions will appear on this site.

However, once a version of The Obvious NDA appears on this site, that version will always read exactly the same. When we find a way to improve 1.0.0, this site will publish a new version, perhaps version 1.0.1. Version 1.0.0 will remain available.
